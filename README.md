# xboltz.com — E-Bike Clearance Outlet

Static site for clearing used/demo e-bikes (Snapcycle S1 / R1 / Warrior + Mihogo Air Max). Winter Garden, FL.
Hosted free on GitHub Pages, custom domain xboltz.com (GoDaddy DNS).

- Edit prices/products directly in `index.html` (search the bike name).
- Videos: replace `.vph` placeholders in the Videos section with YouTube embeds.
- No payment links by design — in-person payment only.
